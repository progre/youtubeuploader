# YouTube Auto Uploader
[![Build Status](https://travis-ci.org/progre/youtubeuploader.svg?branch=master)](https://travis-ci.org/progre/youtubeuploader)


This is a tool for live-caster to automatically uploaded to YouTube
from time to time the recording of the encoder, such as OBS.

Since the uploaded recording become private setting,
you can decide whether or not the public at a later time.

Since the uploaded recording enters into the trash,
you can restore if you do not like the result that was raised to YouTube.

How to use
----

### Preparation

1. Unzip to the appropriate folder, and then start
2. Click icon and "Option"
3. It will be in the same location as the path of the recorded video file of the encoder "録画ファイルのパス"
4. To fill the "チャンネル名(Channel name)", "タグ(tag)" appropriately, and close the window
5. Click icon and "Authenticate"
6. Sign in with your Google account

### Use

1. Start live casting
2. When the live casting is finished, it will be uploaded to YouTube


Enviroment
----

- Windows10
- OS X 10.10.5
